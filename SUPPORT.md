# Support Guidelines

Use support before opening a formal change when you need clarification about this repository's structure, reuse model, or validation rules.

## Use support for

- Questions about whether a change belongs in the repository instance or in `artifacts/`.
- Questions about how to instantiate a reusable artifact.
- Questions about validator failures or governance workflow behavior.

## Use issues or pull requests for

- Bugs in a reusable artifact.
- Requests for new templates, schemas, policies, or workflow assets.
- Structural changes to the repository model.
