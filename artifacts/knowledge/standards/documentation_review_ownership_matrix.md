## Purpose

Define who owns each document class and who reviews it so that maintenance stays explicit.

## Required fields

- Document class
- Primary owner
- Backup owner
- Review cadence
- Reviewer group
- Publication status

## Rules

- Every public document class must have a named owner.
- Review cadence must be explicit and actionable.
- Ownership changes should be traceable in the decision log.
