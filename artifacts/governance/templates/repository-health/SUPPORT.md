# Support Guidelines

Use support channels for questions that are not ready to become an issue or PR. The `{{SUPPORT_CHANNEL}}` is the public help desk; escalate automation requests through `{{AUTOMATION_CHANNEL}}`.

## When to use support

- Pre-flight questions about governance application in a new repository.
- Notifications that a downstream automation consumes this corpus but produces an error.
- Clarifications around placeholder values such as `{{REPO_NAME}}` or `{{GOVERNANCE_TEAM}}`.

## When to open an issue

Use the issue forms if you are reporting a bug, change request, incident, or investigation need—this repo’s intake surfaces keep the context structured.
