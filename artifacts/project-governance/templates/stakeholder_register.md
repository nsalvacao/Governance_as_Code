# Stakeholder Register

## Purpose

Track the people or groups that influence, sponsor, use, or review the project.

## Fields

- stakeholder name
- role or group
- interest
- influence
- engagement stance
- communication needs
- owner

## Usage

1. Keep the register current as the project evolves.
2. Use consistent categories so engagement plans can be generated automatically.
3. Cross-link each stakeholder to the communication plan.
