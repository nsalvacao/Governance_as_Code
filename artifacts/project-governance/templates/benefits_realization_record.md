# Benefits Review / Benefits Realization Record

## Purpose

Track whether expected benefits were achieved and sustained.

## Fields

- benefit name
- expected measure
- baseline
- target
- actual result
- review date
- owner
- follow-up action

## Usage

1. Link each benefit to the original business case.
2. Record actual performance against the agreed target.
3. Note whether the benefit is temporary, ongoing, or not realized.
