# Status / Highlight Report

## Purpose

Provide a periodic, concise view of progress, health, and deviations.

## Fields

- reporting period
- overall status
- accomplished work
- upcoming work
- key risks
- key issues
- decisions needed

## Usage

1. Keep it short enough for routine review.
2. Prefer measurable status statements over narrative summaries.
3. Make exceptions and blockers explicit.
