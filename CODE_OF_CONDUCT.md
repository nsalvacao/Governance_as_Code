# Code of Conduct

This repository expects respectful, constructive, and publication-safe collaboration from both human contributors and AI-assisted workflows.

## Expected behaviour

- Be precise, respectful, and evidence-based.
- Separate disagreement about design from criticism of people.
- Avoid introducing public content that exposes private rationale or sensitive operational detail.

## Reporting

1. Use [`.github/ISSUE_TEMPLATE/investigation_request.yml`](./.github/ISSUE_TEMPLATE/investigation_request.yml) for governance or process concerns.
2. Use [SUPPORT.md](./SUPPORT.md) for clarification before filing a formal change request.
3. Keep sensitive details out of public issues when private handling is more appropriate.

## Enforcement

The repository maintainer may request edits, pause discussion, or close contributions that do not meet the collaboration standard of this repository.
